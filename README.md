# Extension Blocker Assignment MVP

This project implements a minimal MVP for the server assignment:
"파일 확장자 차단".

## Stack
- Java 21
- Spring Boot
- Spring Data JPA
- Flyway
- H2 (local run)
- PostgreSQL (integration test / deploy target)
- Vanilla JS (single page)
